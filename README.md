# drrs-be
Contains api for backend features. You can use the api by downloading **Postman**, then add the url endpoints and the JSON for input.

## Local installation
1. Set up `.env' file
2. `npm install`
3. `npm run dev`
4. Use **Postman** and then add the `URL` and `JSON` inputs.
